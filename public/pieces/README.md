## Chess Pieces Directory

This directory should contain the chess piece images in the following structure:

```
pieces/
├── default/
│   ├── wb.png  (white bishop)
│   ├── wk.png  (white king)
│   ├── wn.png  (white knight)
│   ├── wp.png  (white pawn)
│   ├── wq.png  (white queen)
│   ├── wr.png  (white rook)
│   ├── bb.png  (black bishop)
│   ├── bk.png  (black king)
│   ├── bn.png  (black knight)
│   ├── bp.png  (black pawn)
│   ├── bq.png  (black queen)
│   └── br.png  (black rook)
├── modern/
│   └── (same structure as default)
└── classic/
    └── (same structure as default)

Please add your chess piece images in PNG format with transparent backgrounds.
Each piece should be named according to its color (w/b) and type (k/q/r/b/n/p).
